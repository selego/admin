# SELEGO Boilerplate

## Setup admin

- cd admin
- Run `npm i`
- Run `npm run dev`
- Set variables such as `apiURL`, `S3PREFIX` and `SENTRY_URL` in `config.js` file
- Signup as new user and continue

## Setup api

- cd api
- Run `npm i`
- Run `npm run dev`
- Set project variables in `.env` (create new if not found) and `config.js` file
